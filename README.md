# homebrew-iosevka

Personal Homebrew tap for some [Iosevka](https://typeof.net/Iosevka/) variants. Versions track [upstream releases](https://github.com/be5invis/Iosevka/releases/latest).

## Install

```bash
brew tap iningz/iosevka
brew install --cask iningz/iosevka/font-iosevka-ss04
brew install --cask iningz/iosevka/font-iosevka-slab-ss04
brew install --cask iningz/iosevka/font-iosevka-term-ss04-nerd-font
brew install --cask iningz/iosevka/font-iosevka-aile
brew install --cask iningz/iosevka/font-iosevka-etoile
```
